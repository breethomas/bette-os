# Strategy Directory

This directory contains strategic thinking, analysis, and planning documents.

---

## Context to Load

**Always read first:**
- `initiatives/{Strategy Framework}` - Your strategic foundation

---

## Claude's Role

**Thought partner and editor for strategic work.**

Internal strategy work has more latitude than published content. Claude can draft and rewrite under direction. Speed and clarity matter more than voice ownership.

For all work:
- Test ideas and highlight alternative points of view
- Catch gaps in logic and reasoning
- Challenge assumptions with evidence
- Connect dots across sources and sessions

---

## Core Workflows

### Synthesize transcripts into insights
1. Read the raw transcript
2. Extract: key decisions, open questions, strategic implications, action items
3. Connect findings to the strategic framework
4. Flag contradictions or tensions with existing strategy
5. Output: structured synthesis document

### Draft and refine strategic docs
1. Read the strategic framework for context
2. Understand the document's purpose and audience
3. Draft with clear structure: problem, analysis, recommendation
4. Multiple review rounds: content > structure > accuracy > typos

### Research and analysis
1. Start with: What do we know? What don't we know?
2. Use available sources (Notion, Linear, meeting notes)
3. Distinguish clearly: facts vs. analysis vs. opinion
4. Surface gaps that need investigation
5. Present findings with evidence, not just conclusions

### Prep presentations/updates
1. Identify the audience and what they need to decide or understand
2. Pull from existing strategy docs (don't recreate from scratch)
3. Format for consumption: exec summary, key points, supporting detail
4. Every point should connect to the strategic framework

## Directory Structure

- `initiatives/` - Strategic documents and working drafts
- `context/` - Background research, market analysis (create as needed)
- `raw-sources/` - Meeting transcripts, external documents, inputs (create as needed)
- `_sessions/` - Claude session logs and synthesis notes (create as needed)

---

## The Filter

When analyzing or recommending, always ask:
1. Does this align with the strategic framework?
2. Does this support a core value proposition?
3. Is this accelerate, protect, or wait?
4. What's the evidence? What's still assumption?

---

## Session Management

Strategy sessions can run long. After completing major synthesis or document drafts:
- Suggest saving session notes to `_sessions/`
- Capture: what was produced, key decisions, open questions, next steps
