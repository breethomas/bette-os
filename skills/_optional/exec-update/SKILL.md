# Draft Leadership Update

Draft a status update for a recurring leadership meeting.

## Context Files

Read these first:
- `{Home Directory}/BACKLOG.md` — what's been happening
- `{Home Directory}/GOALS.md` — quarterly priorities
- `{Strategy Directory}/initiatives/{Strategy Framework}` — strategic framework

## Workflow

### 1. Gather context

Run the weekly review thought process:
1. What got done this week?
2. What didn't get done? Why?
3. What's coming next week?
4. Are current priorities still aligned with GOALS.md?

### 2. Format into four sections

- **Top 3**: What moved on priorities. Be specific. Tie to strategic outcome. Exactly 3 — forces prioritization.
- **Insight**: One thing learned that changes thinking or validates/challenges an assumption
- **Decision/Escalation** (if any): What needs leadership awareness or input
- **Blocker** (if any): What's stuck and what specific help is needed

### 3. Quality checks

- Every bullet MUST connect to the strategic framework
- No activity reports ("had meetings", "working on X") — only progress, insights, decisions
- Keep each bullet to 1-2 sentences max

### 4. Output

```
## Update — [Date]

### Top 3
1. [What moved] — [strategic connection]
2. [What moved] — [strategic connection]
3. [What moved] — [strategic connection]

### Insight
[One thing learned that changes thinking]

### Decision/Escalation
[What needs leadership input, or "None this week"]

### Blocker
[What's stuck + specific help needed, or "None this week"]
```

## Output Style

- Concise — summary row, not essay
- Strategic — tied to framework, not activity
- Honest — if nothing moved, say so and explain why
